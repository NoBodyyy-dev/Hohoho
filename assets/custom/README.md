# Custom-модели (стиль Meccha Chameleon)

Сюда кладём .glb, сгенерённые нейронкой под чистый мультяшный стиль игры.
Godot подхватит их автоматически: `ModelLib.place(parent, "c:<имя_файла_без_glb>", ...)`.
Если файла нет — код падает на процедурную геометрию, игра не ломается.

Требования к модели:
- формат **.glb** (бинарный glTF, с вшитыми текстурами)
- один объект, дно у нуля, смотрит по +Z
- low-poly, чистые формы, без мелкой детализации
- масштаб не важен — ModelLib сам подгонит под габарит

После добавления файлов: открыть проект в Godot один раз (импорт), или
`Godot --headless --path . --import`.
