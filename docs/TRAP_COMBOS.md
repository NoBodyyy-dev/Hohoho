# Связки ловушек — 100+ идей

Формат: **триггер → эффект** (+ статус). Игрок собирает их сам из предметов
(триггер / эффект / связь-провод) на любых поверхностях. Часть — «именованные
связки» с бонусом стиля, часть открывается игроками как секреты.

Легенда эффектов: 🐌 замедление · 💫 оглушение · 🌀 головокружение ·
🙈 ослепление · 🔥 огонь · ⚡ электрошок · 💧 мокрый · ↩️ отлёт · 🪤 захват.

## Как это ложится на код

Всё выражается через тройку **Trigger / Effect / Link** (уже в игре):
- Триггеры: растяжка, плита, датчик двери/окна/ящика, приманка, таймер, провод.
- Эффекты: падение любого RigidBody-пропса, ведро (вода/краска/мука/гвозди),
  шокер, масло/клей/скольжение, петарда/огонь, сеть/капкан, дым/ослепление.
- Связи: провод/верёвка соединяют любой триггер с любым эффектом на расстоянии,
  каскадируются в цепи (ДОМИНО).

Новые эффекты, которых пока нет в `Defs.TRAPS` (добавить по мере ассетов):
краска, мука, гвозди-ведро, утюг, гиря-маятник, капкан, яма-люк, дым-шашка,
батарея, стул-подстава, диван-катапульта, вращающийся шкаф.

Приоритет добавления — по наличию моделей и по «вау»-эффекту на видео.
