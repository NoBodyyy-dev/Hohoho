# Santa Ho Ho 🎅🪤

Кооп «Один дома»: вы — мелкие пиздюки, которые защищают дом от **грабителей**.
Грабители — тоже живые игроки (1 грабитель на 4 мелких). Их цель — найти спрятанные
драгоценности до приезда полиции. Ваша — превратить дом в полосу препятствий.

> ⚠️ Идёт пивот концепции с «поймай Санту» на «грабители». ИИ Санты и процедурный
> генератор особняка удалены. Следующие шаги: контроллер грабителя (на базе santa.gd),
> драгоценности в мебели, фаза «2 мин подготовка / 4 мин до полиции»,
> GLTF-модели (KayKit/Quaternius + Mixamo) вместо процедурных фигурок.

Движок: **Godot 4.5**, вид от **первого лица**, яркий low-poly (вайб RV There Yet?).

Графика: ассеты **Kenney** (CC0) — Furniture Kit + Holiday Kit в `assets/kenney/`.
Загрузка через `scripts/world/model_lib.gd` (нормализует масштаб, ставит на пол,
вешает коллайдер); если модели нет — генератор падает обратно на процедурные кубы.

## Запуск

1. Положи Godot 4.5 в `tools/` (папка в .gitignore) → Import → папка проекта → Play (F5).
2. Смоук-тест: env `SANTA_TEST=1` headless. Скриншоты: `SANTA_SHOT=<папка>`.

## Управление

| Клавиша | Действие |
|---|---|
| WASD + мышь | бег и камера (от первого лица) |
| SHIFT | спринт |
| **E (держать)** | мелкий: заколотить окно / рыться в мебели · грабитель: обыск / взлом окна / связать |
| Пробел | прыжок / выпутываться из верёвок / QTE установки |
| **F** | режим ловушек: хотбар снизу, призрак-превью на клетке |
| 1–9 / колесо | выбор предмета в хотбаре |
| ЛКМ | поставить ловушку (канал + QTE). Растяжка — **две точки**: клик-конец, тянешь нить, клик-конец. **Провод** — клик по своей ловушке, потом по люстре/шкафу/ТВ/ковру: связка на расстоянии |
| ПКМ | отмена установки / сброс конца растяжки |
| Q / R / F | за грабителя: глаз-алмаз (ловушки) / рык / чутьё наживы |
| Esc | пауза / выйти из режима ловушек |

## Механики

- **Лобби**: кто Санта (бот / я / рандом), время подготовки (30–90 c), время Санты (3–6 мин).
- **Комбо с обстановкой**: верёвка+люстра (роняет люстру), ведро/клей — над дверью/на пороге,
  масло+плитка в ванной (каток), петарда+камин (блок дымохода), всё на ковре — почти невидимо.
- **Комбо ловушка+ловушка**: соседние клетки срабатывают **цепочкой**, установка рядом с другой
  ловушкой даёт бонус к качеству. Печенье-приманка тянет Санту в нужную клетку.
- **Кинетика (пинг-понг)**: **Шарики** толкают Санту ВПЕРЁД по инерции, **Банан** — швыряет
  НАЗАД. Поставь шарики, а за ними банан — Санта уедет вперёд на банан и улетит обратно на шарики.
- **Триггеры с задержкой**: **Нажимная плита** и **растяжка** рядом с люстрой/шкафом «заряжают»
  падение объекта. Колесом мыши при установке настраиваешь задержку (0–3 с): Санта наступает,
  идёт таймер, и люстра/шкаф рушится ровно туда, где он будет через N секунд. Промазал по таймингу —
  грохнется мимо.
- **Взаимодействие с домом**: в режиме F над люстрами/шкафами/камином/**ТВ**/**холодильником**/**коврами**
  всплывают подсказки, а клетки рядом подсвечиваются янтарём. Целься прямо в люстру — сетка ловит и её.
  - **Телевизор** — триггером бьёт Санту током (электрошок + головокружение).
  - **Холодильник** — при завале разливает молоко: пол скользкий И под белым молоком **не видно ловушек**.
  - **Ковёр** — растяжка/плита на ковре **выдёргивает** его из-под Санты (падение + отлёт).
- **Статусы**: у ловушек в списке видно, что они дают (стан, замедление, головокружение,
  дезориентация, отлёт). Активные эффекты Санты показаны справа сверху.
- **Головокружение/дезориентация** (духи): Санту шатает, он не может сажать в мешок и **забывает**
  все примеченные ловушки — идёт напролом.
- **Скрытые связки** (нигде не описаны — догадайся сам): мокрый после ведра + гирлянда-шокер =
  мегавольт; петарда рядом с масляной лужей = **пожар**; печенье вплотную к клею = Санта прилипает
  прямо за столом.
- **ХАОС-КОМБО**: попадания подряд (окно ~4.5 с) складываются в цепь с растущим множителем
  (КОМБО → ХАОС → БЕЗУМИЕ → ЛЕГЕНДАРНЫЙ ХАОС) и копят «стиль». **Именованные связки**
  (КАРУСЕЛЬ, МОКРОЕ ДЕЛО, ПРЕИСПОДНЯЯ, ГРАВИТАЦИЯ…) дают бонус — часть открыта, часть секретна.
  Накопленный стиль в конце матча конвертируется в **монеты**: чем изобретательнее машина —
  тем больше награда. Цепи и множитель поощряют кооп-пати строить всё более безумные комбинации.
- **QTE**: зафейлил установку — ловушка может дать осечку.
- **Санта = игрок с асимметрией**: он **раскладывает** подарки (не крадёт), и **ловушки ему не видны**.
  Пацаны видят золотые метки мест доставки (знают, где Санта разложит подарки → куда ставить ловушки),
  а Санта их **не видит** — он не знает, где раскладывать. Умения Санты:
  - **F — чуйка на подарки**: всплывают ПРИМЕРНЫЕ столбы света у неразложенных мест (со сдвигом,
	не точно) — «где-то здесь». Плюс «горячо/холодно»: вблизи места метка теплеет.
  - **Q — чуйка на ловушки**: подсвечивает скрытые ловушки рядом (единственный способ их увидеть).
  - **R — ХО-ХО-ХО**: пугает пацанов, срывает им установку. **SHIFT** — рывок.
  В последнюю минуту Санта **звереет**. Бота нет — за эту сторону всегда играет человек.
- **Поимка**: пока Санта оглушён — стой рядом, качается шкала. Полная — Санта в ваших руках.

## Персонажи

- **Шустрик** — скорость +30%. **Умник** — эксклюзив Сеть и Гирлянда-шокер, карманы 8 оч.
- **Крепыш** — карманы 12 оч., сам вылезает из мешка (1 раз). **Мелкая** — установка x0.6, скрытнее.

Опыт → уровень (быстрее установка, +очки карманов на 3/6/9). Монеты → скины и перк.

## Локации

- **Дача у Лёхи** (24×14): камин-дымоход, две люстры, кабинет, веранда, 5 входов.
- **Квартира Макса** (20×12): балкон вместо камина, детская, кабинет, плитка в ванной, куча дверей.

## Структура кода

```
scripts/autoload/defs.gd       — ВСЕ данные: персонажи, предметы, ловушки, локации, тайминги
scripts/autoload/save_game.gd  — прогрессия (монеты, xp, скины)
scripts/main.gd                — меню ↔ матч, смоук-тест и автоскриншоты
scripts/game/match.gd          — фазы, цепные комбо, поимка, награды, эффекты
scripts/world/house_builder.gd — генерация дома (мебель, окна, гирлянды) + навграф + ночное небо
scripts/world/minifig.gd       — фигурки с анимацией ходьбы, предметы в руках
scripts/player/kid.gd          — FP-контроллер (качка, спринт, руки)
scripts/player/trap_mode.gd    — режим F от первого лица
scripts/santa/santa.gd         — FP-контроллер Санты/грабителя (Q/R-фишки, статусы)
scripts/ui/                    — тема, меню с 3D-фоном, HUD
```

## Дорожная карта до Steam

1. **Мультиплеер**: Godot ENet → GodotSteam (лобби уже в UI; Санта — живой игрок, до 4 пацанов).
2. Командные ловушки на двоих (двое держат верёвку — двойное QTE).
3. Локация-особняк с 2 этажами, звук, больше анимаций (прыжок в мешок, разморозка).
4. Steamworks: достижения, скины-дропы за поимку.
